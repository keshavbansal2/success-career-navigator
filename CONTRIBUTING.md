# Contributing to CareerNav AI

Thank you for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/success-career-navigator.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes and commit them
5. Push to your fork and open a Pull Request

## Development Setup

Follow the [Quick Start](README.md#quick-start-local-development) guide in the README.

## Code Style

- **Backend (Python)**: Follow PEP 8. Use type hints where possible.
- **Frontend (TypeScript)**: Use functional components and React hooks. Keep components small and focused.

## Commit Message Format

Use conventional commits:
- `feat:` — new feature
- `fix:` — bug fix
- `docs:` — documentation changes
- `chore:` — maintenance tasks
- `refactor:` — code refactoring

## Reporting Issues

Open a GitHub Issue with:
- A clear title and description
- Steps to reproduce (for bugs)
- Expected vs actual behavior
